CRYSTAL VISUAL DESIGN SYSTEM

BACKGROUND & ROLE

Crystal is a new email technology based on empathy. Through the use of public information, the program aims to help users communicate more effectively, and efficiently, by providing email, speaking and selling tips tailored to recipients preferred communication styles and personalities.

After examining the user onboarding process that Crystal is currently using to convert interested users into full time paying customers, the apprenticeship team at Fresh Tilled Soil was tasked with creating a visual design to move the product to the next level.

With the technology being a priority from the start, the visual design was a secondary focus when creating the product.

I created the design system using Photoshop and presented it to the finder for review. I also continued to be the client's main point of contact as I, and other team members, navigated the project's deadlines.

VISUAL DESIGN SYSTEM- COLOR & STATES

In order to create a visual asthetic that can be manipulated as the site, and product grows, I created a visual system using modular content elements that can be used for multiple types of content and moved/removed as needed.

Colors were pulled from the logo, and supplemented with complementary blues and a tertiary orangy red for emphasis (alerts and completed actions). Different shades and stroke widths are used to signal button and input states.

VISUAL DESIGN SYSTEM- TYPE & GRID SYSTEM

In order to create a feeling of openness and trustwortyness, Lato (a Google Webfont) was chosen. Its various weights provide options for emphasis, headlings, body text, etc.

A twelve column grid grounds the design and allows for easy scalability to medium and small screens, or application design if that avenue is pursued in the future.
